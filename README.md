# 一份小火箭基础配置
包含RuleSet，策略组以及部分脚本（脚本或需配合BoxJS使用）。

如需使用自定义 GeoIP 文件
建议在「设置」里「GeoLite2数据库」的「URL」填写使用「 https://raw.githubusercontent.com/Loyalsoldier/geoip/release/Country.mmdb 」 会自动更新

PS: 设置了自定义 GeoIP 文件后或可把GeoIP,CN,DIRECT规则删除。
